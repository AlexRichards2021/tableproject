Tables CheatSheet!

Here Is alex's how to on creating and styling tables using html and css

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Header Creates a table head
<th> Table Head Data Creates data in a table head
<tbody> Table Body Creates a table Body
<tr> Table Row Creates a Table Row
<td> table data The Table Row Container
<tfoot> Table Footer Creates a table footer
<colspan> Column Span States how many columns a td element should span
<rowspan> Row Span States how many rows a td element should span

Tag Name Example
{border} Border properties To Adjust border properties, ie 1px solid black, to add a solid black border
{width} Width To adjust how much of the screen the table spans, ie 100% for full screen
{Table-layout:fixed} Fixes Table Layout Makes the table behave more predictably, making the data easier to read and style
{thead th:nth-child(x): width: x%} selects table heading to adjust column of by adjusting child(x) you can select a column to adjust heading of, and adjusting width(x) to dictate how much of the table that column takes up
